# todoc

**todoc** es una pequeña herramienta en C para gestionar notas rápidas directamente desde la terminal.
Está pensada para ser simple, rápida y sin dependencias externas, ideal para flujos minimalistas en Linux.

---

## ✨ Características

* 📌 Añadir notas desde la terminal
* 📖 Leer notas guardadas
* 🧹 Limpiar pantalla
* ⚡ Ligero y rápido
* 🧠 Código simple en C (ideal para aprender)

---

## 🚀 Uso

Al ejecutar el programa:

```bash
make run
```

Verás el menú:

```
[a]dd - [r]ead [c]lear - [q]uit
>
```

### Opciones:

* `a` → Añadir una nueva nota
* `r` → Leer todas las notas
* `c` → Limpiar pantalla
* `q` → Salir
